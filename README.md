# Neural Network Models & Simulations | 類神經網路實作與模擬

## 📝 About The Project (關於專案)
本專案收錄了類神經網路 (Artificial Neural Networks, ANN) 相關的演算法實作與模擬分析報告。內容主要探討不同神經網路架構的數學模型、學習機制及其應用領域，包含前饋型網路 (Feedforward Neural Network) 與回饋型網路 (Recurrent Neural Network) 的分析與測試。

此專案旨在展現對於機器學習底層演算法的理解、系統模擬能力以及數據分析技巧。

## 🔬 Project Contents (專案內容)

本專案包含以下三個主要的類神經網路研究與模擬單元：

* **Backpropagation Algorithm (BP 倒傳遞演算法)**
  * 涵蓋基礎多層感知機 (MLP) 架構的建立與模擬測試。
  * 驗證網路的訓練過程、權重更新機制 (Backpropagation) 以及不同參數設定 (如學習率) 下的收斂效能評估。

* **Radial Basis Function Network (RBF 徑向基底函數網路)**
  * 實作基於徑向基底函數的三層前饋型神經網路。
  * 應用於非線性函數近似與模式分類，探討隱藏層神經元中心的選取、平滑參數設定對整體網路訓練效率與準確度的影響。

* **Hopfield Network (霍普菲爾德神經網路)**
  * 探討具備聯想記憶 (Associative Memory) 功能的單層回饋型神經網路。
  * 實作記憶儲存與模式提取，分析網路的能量函數 (Energy Function)、收斂特性以及對雜訊的容忍與還原能力。

## 🛠️ Tools & Technologies (開發工具與技術)
* **核心理論**: 倒傳遞學習、函數近似、聯想記憶、神經網路優化演算法
* 使用的程式語言: MATLAB, Python
